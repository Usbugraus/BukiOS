# BukiOS
**BukiOS 1.0.5** - A Terminal Based Operating System. 
Coded with Python **3.10.11**

## Features
* Markdown editor
* Python editor
* Text editor
* Recovery mode
* Auto module install system
## Getting Started
To install and start BukiOS, run:

1.
```bash
git clone https://github.com/yourusername/BukiOS.git
```

2.
```bash
cd  BukiOS
```

3.
```bash
python Bootloader.py
```
